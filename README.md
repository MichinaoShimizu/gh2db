# gh2db

## Usage

create `.env` file.

```bash
GITHUB_TOKEN=
GITHUB_TARGET_ORGANIZATION_NAME=

DB_DIALECT=
DB_USERNAME=
DB_PASSWORD=
DB_HOSTNAME=
DB_PORT=
DB_NAME=
DB_ECHO_TYPE=0
```

```bash
$ python gh2db/gh2db/ --create_database
usage: [-h] [--update_user_repos] [--update_org_repos] [--create_all] [--drop_all] [--delete_all] [--count_all]
```
